# 0x.credit PoC

This is the Vite + React frontend for 0x.credit. It uses Reown AppKit, wagmi, viem, and the local Gearbox SDK checkout in `../sdk`.

## Usage

```sh
npm install
npm run dev
```

Set `VITE_REOWN_PROJECT_ID` in `.env.local` to use a specific Reown project.
